**PozdravlyatorGUI** 🎉

Простое настольное приложение для Windows на C# и WinForms с Material Design-интерфейсом для хранения и отслеживания дней рождения.

**Возможности**

  Добавление новых записей о днях рождения, 
  Редактирование существующих записей, 
  Удаление записей с подтверждением, 
  Автоматическое сохранение данных в JSON-файл, 
  Отображение возраста человека, 
  Подсчёт количества дней до следующего дня рождения, 
  Просмотр всех записей, 
  Фильтрация именинников на сегодня, 
  Просмотр ближайших дней рождения (в течение 30 дней), 
  Уведомление при запуске программы о сегодняшних именинниках, 
  Современный интерфейс на базе MaterialSkin.

**Используемые технологии**

  C#

  .NET Windows Forms
  
  MaterialSkin
  
  JSON Serialization (System.Text.Json)

**Хранение данных**

  Все данные сохраняются локально в файл birthdays.json, поэтому информация не теряется после закрытия программы.

**Структура проекта**

  BirthdayEntry — модель записи дня рождения
  
  BirthdayService — работа с данными и JSON-хранилищем
  
  Form1 — графический интерфейс приложения
  
  birthdays.json — файл хранения записей
